# 라이어 게임

실시간 멀티플레이어 웹 기반 라이어 게임입니다. PeerJS를 사용한 P2P 연결로 별도의 서버 없이 플레이가 가능합니다.

## 게임 방법

1. 닉네임을 입력하고 게임에 참여합니다.
2. 2명 이상 모이면 방장이 게임을 시작할 수 있습니다.
3. 게임이 시작되면 한 명이 라이어로 선정됩니다.
4. 라이어를 제외한 모든 플레이어는 제시된 단어를 확인합니다.
5. 돌아가면서 단어에 대해 설명합니다.
6. 모든 플레이어가 설명을 마치면 라이어를 투표합니다.
7. 라이어가 발견되면 라이어는 단어를 맞출 기회를 얻습니다.

## 게임 모드

### 기본 모드
- 한 명의 라이어만 존재합니다.

### 스파이 모드
- 라이어 1명과 스파이 1명이 있습니다.
- 스파이는 단어를 알고 있지만, 정체를 숨겨야 합니다.

## 기술 스택

- HTML, CSS, JavaScript
- PeerJS (WebRTC 기반 P2P 통신)

## 로컬에서 실행하기

1. 리포지토리를 클론합니다.
2. 웹 서버를 실행합니다.
   ```
   npx http-server
   ```
3. 브라우저에서 http://localhost:8080 으로 접속합니다.
